# #4 Web界面 — 爬虫采集功能区（Streamlit Tab1）

**状态**：待开始 | **类型**：AFK | **阻塞于**：#2 | **GitHub**：[#4](https://github.com/linhuanwen/wb-content-tool/issues/4)

## What to build

在 Streamlit Web 界面上实现"功能区1：爬虫采集"Tab。

端到端行为：
1. 用户打开 Web 页面，选择"爬虫采集" Tab
2. 上传包含 asin 列的 Excel 文件
3. 系统自动识别 asin 列，显示"检测到 X 个 ASIN"
4. 点击"开始采集"按钮
5. 实时进度条显示采集进度（已完成 X/Y）
6. 采集完成后在页面上以表格预览结果
7. 点击"下载处理前表格"按钮，导出为标准格式 xlsx 文件
8. 采集失败的 ASIN 在日志区域显示警告
9. 日志区域显示采集过程中的关键信息

## Acceptance criteria

- [ ] 上传非 Excel 文件时给出友好提示
- [ ] 上传 Excel 但无 asin 列时给出提示
- [ ] 能成功采集至少 3 个 ASIN 并在页面预览
- [ ] 下载的 xlsx 文件与"爬虫表格（处理前）"格式一致
- [ ] 进度条实时更新，不卡死 UI
- [ ] 侧边栏可配置无头模式和请求间隔
