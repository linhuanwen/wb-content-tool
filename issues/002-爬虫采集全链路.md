# #2 爬虫采集全链路：ASIN输入 → Playwright爬取 → 处理前表格输出

**状态**：待开始 | **类型**：AFK | **阻塞于**：#1 | **GitHub**：[#2](https://github.com/linhuanwen/wb-content-tool/issues/2)

## What to build

实现亚马逊商品信息爬虫模块，输入包含 ASIN 列的 Excel 文件，输出"爬虫表格（处理前）"。

端到端行为：
1. 读取输入 Excel，自动识别 asin 列
2. 使用 Playwright 异步 API 逐个访问 `https://www.amazon.com/dp/{ASIN}`
3. 提取三个字段：标题（`#productTitle`）、图片 URL（`#altImages` 下所有高清图，用 `;` 拼接）、详情（`#feature-bullets` + `#productDescription`）
4. 反爬策略：随机 User-Agent、3~8 秒随机延迟、模拟滚动、验证码检测并报错、失败自动重试 2 次
5. 输出 Excel 包含 [asin, 图片url, 标题, 详情] 四列，格式与现有"爬虫表格（处理前）"一致
6. 提供 CLI 入口：`python crawler.py --input asin列表.xlsx --output 处理前.xlsx`
7. 提供 `async def crawl_asins(asins, progress_callback=None)` 函数供 Web 界面调用

## Acceptance criteria

- [ ] 用样本 ASIN（B0GVYXC124）测试，能成功提取标题、多图 URL（`;`分隔）、详情
- [ ] 多图 URL 数量 ≥ 2 个（非单图产品）
- [ ] 遇到验证码页面时抛出清晰异常，不静默失败
- [ ] CLI 命令可正常运行并输出 xlsx 文件
- [ ] 输出表格列名和格式与"爬虫表格（处理前）"样本一致
